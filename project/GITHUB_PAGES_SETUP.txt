📋 STEP-BY-STEP SETUP:

1️⃣ CREATE GITHUB ACCOUNT:
   - Go to github.com
   - Click "Sign up"
   - Choose username (example: john-edumcq)
   - Use free account

2️⃣ CREATE NEW REPOSITORY:
   - Click green "New" button
   - Repository name: "edumcq-app" 
   - Make it PUBLIC (required for free)
   - Check "Add a README file"
   - Click "Create repository"

3️⃣ UPLOAD YOUR APP FILES:
   - Click "uploading an existing file"
   - Upload ALL these files from your project:
     ✅ index.html
     ✅ style.css
     ✅ app.js
     ✅ data.js
     ✅ manifest.json (NEW - for mobile install)
     ✅ sw.js (NEW - for offline work)
     ✅ All other files (.bat, .txt, etc.)

4️⃣ ENABLE GITHUB PAGES:
   - Go to "Settings" tab
   - Scroll to "Pages" section
   - Source: "Deploy from a branch"
   - Branch: "main"
   - Folder: "/ (root)"
   - Click "Save"

5️⃣ GET YOUR WEBSITE:
   - Your URL: https://yourusername.github.io/edumcq-app
   - Takes 5-10 minutes to go live
   - Test on phone and computer!

📱 MOBILE INSTALL FEATURE:
Once online, users can:
1. Open your website on their phone
2. Browser will show "Install" option
3. Tap "Install" or "Add to Home Screen"
4. App appears on phone like real app!
5. Works offline after installation!

🔄 TO UPDATE YOUR APP:
