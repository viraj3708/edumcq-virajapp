## üéØ HOW TO USE

### LOGIN:
1. Enter your name and email
2. Click "Start Learning"

### NAVIGATION:
1. Select your Board (CBSE/State)
2. Choose your Class
3. Pick a Subject
4. Select a Chapter
5. Explore through MCQs, Videos, and Summaries

### TAKING QUIZZES:
1. Click on any MCQ Set
2. Read the question carefully
3. Select your answer
4. If wrong, correct answer is shown immediately
5. Click "Next" to continue
6. View results at the end

### RETRYING QUIZZES:
1. Return to MCQ tab
2. See your previous attempts with scores
3. Click "Retry Quiz" to try again

## üÜò TROUBLESHOOTING

**Issue: App not opening**
- Make sure all files are in the same folder
- Try double-clicking index.html instead

**Issue: Progress not saving**
- Make sure to complete quizzes fully
- Don't close browser during quiz

**Issue: Dark mode not working**
- Refresh the page after toggling
- Clear browser cache if needed
