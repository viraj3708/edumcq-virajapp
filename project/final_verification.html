Answer Coloring Implementation

This page explains how the answer coloring feature should work in the EduMCQ application:

How It Works

When a user selects an answer:

  1. The selected option gets a selected class
  2. If the answer is correct, it gets a correct class (green)
  3. If the answer is incorrect, it gets an incorrect class (red)

Expected CSS Classes

/* Correct answer styling */ .option.correct { background: linear-gradient(135deg, #48bb78, #38a169) !important; border-color: #2f855a !important; color: white !important; box-shadow: 0 4px 15px rgba(72, 187, 120, 0.4) !important; } /* Incorrect answer styling */ .option.incorrect { background: linear-gradient(135deg, #e53e3e, #c53030) !important; border-color: #b71c1c !important; color: white !important; box-shadow: 0 4px 15px rgba(229, 62, 62, 0.4) !important; }

Fixed selectOption Function

The updated function in app.js now properly handles class application:

selectOption(selectedIndex) { // Remove any existing selection classes from all options const allOptions = document.querySelectorAll('.option'); allOptions.forEach(option => { option.classList.remove('selected', 'correct', 'incorrect'); }); // Check if the selected answer is correct const currentQuestion = this.currentQuiz[this.currentQuestionIndex]; const isCorrect = selectedIndex === currentQuestion.correctAnswer; // Add appropriate class based on correctness const selectedOption = document.querySelector(`[data-index="${selectedIndex}"]`); if (selectedOption) { selectedOption.classList.add('selected'); if (isCorrect) { selectedOption.classList.add('correct'); } else { selectedOption.classList.add('incorrect'); } } // Store the user's answer this.userAnswers[this.currentQuestionIndex] = selectedIndex; document.getElementById('next-btn').disabled = false; }
