STEP 1: Create GitHub Account
---------------------------
1. Go to https://github.com
2. Click "Sign up"
3. Follow the registration process
4. Verify your email address

STEP 2: Create New Repository
---------------------------
1. Click the "+" icon in top-right corner
2. Select "New repository"
3. Repository name: "edumcq" (or any name you prefer)
4. Set to "Public" (required for GitHub Pages)
5. DO NOT initialize with README
6. Click "Create repository"

STEP 3: Upload Your Files
-----------------------
1. Click "Uploading an existing file"
2. Drag and drop ALL files from your project folder:
   - index.html
   - app.js
   - data.js
   - style.css
   - manifest.json
   - sw.js
   - All image files
   - All batch files
3. Scroll down and click "Commit changes"

STEP 4: Enable GitHub Pages
-------------------------
1. Click "Settings" tab (gear icon)
2. Scroll down to "Pages" section
3. Under "Source", select "Deploy from a branch"
4. Select "main" branch and "/ (root)" folder
5. Click "Save"
6. Wait 1-2 minutes for deployment

STEP 5: Access Your Live Website
------------------------------
1. Look for the green "‚úì" indicator
2. Your website URL will appear: https://YOUR_USERNAME.github.io/edumcq/
3. Click the link to view your live educational app!
